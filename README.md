## to run the project:
-----------------------
1. Clone the project:
  git clone https://github.com/chaaladhif/chaaladhif.git

2. Go to the project directory
  cd P12-SportSee

3. Install dependencies
  npm install

4. Start the Backend
  cd backend
  yarn dev
5. Start the frontend
  cd frontend
  npm start
